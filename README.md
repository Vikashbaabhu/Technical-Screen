# Technical-Screen

So, I built the Python function called sort(width, height, length, mass) that dispatches packages to the correct stack for Smarter Technology's robotic arm. It classifies the package as bulky if its volume exceeds 1,000,000 cm³ or any single dimension is 150 cm or more, and heavy if its mass is 20 kg or more. Based on these checks, the function returns STANDARD for normal packages, SPECIAL for packages that are either bulky or heavy, and REJECTED for packages that are both.
